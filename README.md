# Shepard tone  
You can generate shepard tone wav file by this command.  
`python shapard.py --down=False`  
By default, it generates upward Shepard tone. Set --down as True for downward Shepard tone.  
Shepard tone is a sound that sounds as if its pitch were going up/down infinitely.  

You can find examples here:  
[Audio illusion: Shepard tone (upward) - YouTube](https://youtu.be/45yWy-2C4Hk)  
[Audio illusion: Shepard tone (downward) - YouTube](https://youtu.be/JB1JwnSjeFg)
